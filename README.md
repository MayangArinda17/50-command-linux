5.date

6.date +%R

7.date +%x
Pada sistem Linux dan Unix, format %x yang digunakan bersama dengan perintah date adalah untuk menampilkan tanggal dalam format lokal yang disesuaikan dengan preferensi pengaturan regional sistem.

13.head /etc/passwd
Perintah 'head /etc/passwd' digunakan untuk menampilkan beberapa baris pertama dari file '/etc/passwd' pada sistem mirip Unix, termasuk Linux. File '/etc/passwd' berisi informasi tentang akun pengguna pada sistem, dengan setiap baris mewakili entri pengguna.Secara default, perintah 'head' menunjukkan 10 baris pertama file. Oleh karena itu, 'head /etc/passwd' akan menampilkan 10 baris pertama dari file '/etc/passwd'. Ini dapat berguna untuk memeriksa entri awal dalam file dengan cepat.

27./etc
direktori dalam sistem operasi mirip Unix berisi file konfigurasi dan direktori di seluruh sistem

28./home
Direktori '/home' dalam sistem operasi mirip Unix berfungsi sebagai lokasi default untuk direktori home pengguna.


29./root
 pengguna administratif pada sistem operasi mirip Unix dan memiliki hak istimewa yang lebih tinggi.

 
30./run
sistem berkas sementara (tmpfs) yang biasa ditemukan di sistem operasi mirip Unix, termasuk Linux


31./tmp
Direktori '/tmp' dalam sistem operasi mirip Unix, termasuk Linux, adalah direktori lain yang secara khusus ditujukan untuk file sementara


32./usr
Direktori '/ usr' dalam sistem operasi mirip Unix, termasuk Linux, adalah direktori sistem penting yang merupakan singkatan dari "Unix System Resources." Ini memainkan peran penting dalam mengatur berbagai file, program, pustaka, dan dokumentasi terkait sistem


33./var
Direktori '/ var' dalam sistem operasi mirip Unix, termasuk Linux, adalah direktori sistem penting yang merupakan singkatan dari "variabel." Ini digunakan untuk menyimpan file data variabel â€” file yang dapat berubah dalam ukuran dan konten saat sistem beroperasi.

34./bin and /usr/bin
Direktori '/ bin' dan '/ usr / bin' dalam sistem operasi mirip Unix, termasuk Linux, keduanya merupakan lokasi penting di mana binari yang dapat dieksekusi (file program) disimpan. Namun, mereka melayani tujuan yang sedikit berbeda. 


35./sbin and /usr/sbin
Direktori '/sbin' dan '/usr/sbin' dalam sistem operasi mirip Unix, termasuk Linux, adalah lokasi di mana binari administrator sistem (executable) disimpan. Direktori ini memiliki perintah administrasi sistem penting yang biasanya digunakan oleh pengguna root atau administrator sistem. 



36./lib and /usr/lib
Direktori '/lib' dan '/usr/lib' dalam sistem operasi mirip Unix, termasuk Linux, adalah lokasi di mana file library disimpan. Perpustakaan berisi kode yang umum digunakan oleh beberapa program. 


37./lib64 and /usr/lib64
Direktori '/lib64' dan '/usr/lib64', bila ada, khusus untuk sistem 64-bit dan digunakan untuk menyimpan pustaka bersama 64-bit.
